# Diplom_3(vasilii_kasatov)
## Задание 3: Автотесты для UI

### Автотесты для проверки веб-приложения Stellar Burgers

### Реализованные сценарии

Описаны элементы, которые используются в тестах, с помощью паттерна Page Object.

Протестирована функциональность в Google Chrome и Mozilla Firefox.

Отчет: `allure_results`

### Структура проекта

- `locators` - содержит локаторы страницы
- `pages` - содержит элементы, которые используются в тестах
- `tests` - пакет, содержащий тесты, разделенные по классам
- `conftest` - fixture
- `data` - тестовые данные
- `urls` - адреса страниц используемых в тестах
